# Day 3 – Advanced Joins & Subqueries

## FULL OUTER JOIN
Returns all rows from both tables, with NULLs where no match exists.

## CROSS JOIN
Cartesian product of two tables.

## SELF JOIN
Join a table to itself, useful for hierarchical data.

## Subqueries
- Scalar → returns single value.
- Correlated → executes per row of outer query.
- IN/EXISTS → filters based on another query.

## Common Table Expressions (CTEs)
Temporary result sets defined with WITH, useful for breaking down complex queries.
